# Ledger

This directory contains public transparency records.

## Files

- `donations.csv`: incoming donation records.
- `disbursements.csv`: outgoing support records.

## Rules

- Do not include private keys or seed phrases.
- Do not include raw identity documents.
- Do not include unredacted beneficiary private information.
- Use beneficiary codes instead of real names.
- Use tx hashes and evidence references when available.
