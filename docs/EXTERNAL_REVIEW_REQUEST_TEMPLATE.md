# External Review Request Template

Use this template when asking a trusted reviewer to inspect Open Aid Ledger before any donation-ready release.

## Review request

```text
Project: Open Aid Ledger
Release under review: v0.10.0-pre-1.0-candidate-review
Review type: transparency / governance / public safety / legal-tax note / privacy
Donation state: inactive
Wallet publication state: unpublished
```

## Reviewer scope

Please review:

- repository status wording;
- donation activation gates;
- wallet publication procedure;
- campaign lifecycle documentation;
- ledger validation approach;
- beneficiary privacy controls;
- emergency freeze procedure;
- public status page wording;
- whether any page could mislead donors or supported people.

## Out of scope

This review does not ask for:

- custody of assets;
- signing authority;
- trading advice;
- investment review;
- donation activation approval.
