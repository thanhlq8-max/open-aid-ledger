# Status Badge Guide

This project may use badges only if they do not imply donation activation.

## Allowed badge meanings

- CI status;
- documentation status;
- donation inactive;
- public template;
- read-only design.

## Disallowed badge meanings

- donation live;
- verified charity;
- audited fund;
- guaranteed impact;
- investment return;
- trading performance;
- wallet custody safety.

## Recommended wording

```text
Donation status: inactive
Wallets: unpublished
Custody automation: none
```

Do not use badges that imply regulatory approval, charity registration, exchange partnership, or investment performance unless formally verified and documented.
