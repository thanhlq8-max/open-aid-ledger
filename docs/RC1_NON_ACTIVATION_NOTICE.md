# v1.0.0-rc1 Non-Activation Notice

This repository is a transparency and governance template at v1.0.0-rc1.

No donation intake is active in this candidate. No real wallet address is published by this candidate. No maintainer should solicit funds by claiming that v1.0.0-rc1 is an active donation launch.

The correct public message for this release is:

> Open Aid Ledger has reached a donation-ready candidate review stage. The repository remains inactive for donation collection until a separate reviewed activation process is completed.

## Prohibited claims

- Do not claim active collection.
- Do not claim charity registration unless verified outside this repository.
- Do not claim financial return.
- Do not request trading-account funding.
- Do not publish personal information of supported people.
