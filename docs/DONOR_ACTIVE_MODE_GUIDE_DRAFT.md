# Donor Active-Mode Guide Draft

This is a draft donor-facing guide for a possible future active release.

It does not activate donations, publish receiving details, or change project status.

## Current status

```text
DONOR_ACTIVE_MODE_GUIDE: DRAFT
DONATIONS_ACTIVE: NO
WALLETS_PUBLISHED: NO
ACTIVATION_APPROVED: NO
GO_LIVE: NO
```

## Purpose

A future active-mode guide must help donors verify status, asset, network, public records, and freeze state from repository evidence.

## Draft donor checks

Before sending in any future active release, a donor should be able to verify:

- [ ] public dashboard status is active;
- [ ] receiving detail is published from repository source only;
- [ ] asset name is exact;
- [ ] network name is exact;
- [ ] public ledger location is linked;
- [ ] report location is linked;
- [ ] freeze status is visible;
- [ ] last review date is visible;
- [ ] wrong-network warning is visible;
- [ ] no private-message instruction overrides the public source.

## Required review before use

- [ ] Governance reviewer checks wording.
- [ ] Account protection reviewer checks channel warnings.
- [ ] Legal or tax reviewer status is recorded.
- [ ] Public status reviewer checks dashboard alignment.
- [ ] Latest CI result is attached.
