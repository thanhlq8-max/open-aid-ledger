# Repository About Setup

This file records the recommended GitHub repository sidebar settings for Open Aid Ledger.

## Website field

Set the repository Website field to:

```text
https://thanhlq8-max.github.io/open-aid-ledger/
```

This URL is the public dashboard entrypoint.

## Suggested description

```text
Transparency-first public ledger template with a static public dashboard, dry-run evidence loop, and safety-first review gates.
```

## Suggested topics

```text
public-ledger
transparency
open-source
github-pages
dashboard
dry-run
governance
```

## Setup path

```text
Repository -> About gear icon -> Website -> paste public dashboard URL -> Save changes
```

## Operating rule

The Website field is only an access shortcut to the public dashboard. Project status remains controlled by repository files and review gates.
