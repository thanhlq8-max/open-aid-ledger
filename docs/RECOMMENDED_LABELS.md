# Recommended GitHub Labels

Create these labels after publication.

| Label | Purpose |
|---|---|
| `audit` | ledger/report/wallet review |
| `beneficiary-privacy` | privacy-sensitive beneficiary handling |
| `campaign` | campaign proposal or update |
| `documentation` | docs-only work |
| `good first issue` | easy contributor task |
| `help wanted` | community contribution welcome |
| `ledger` | donation/disbursement CSV issue |
| `policy` | governance or policy update |
| `privacy-review` | requires privacy review |
| `public-good` | OSS/community utility funding |
| `security` | security-sensitive issue |
| `support-request` | support request issue |
| `wallet-change` | wallet metadata/address change |
