- Publish public repository.
- Pass validation workflow.
- Keep donations inactive until wallet review is complete.
- Create labels and initial governance issues.

## v0.2.0 — Ledger Safety Foundation

Status: implemented.

- Hardened wallet metadata schema to v1.1.
- Added stronger wallet metadata validation.
- Added stronger ledger CSV validation.
- Added validator regression tests and CI test execution.
- Kept donations inactive and wallets unpublished.

## v0.3.0 — Sample Transparency Report + GitHub Pages Foundation

Status: implemented.

- Added sample transparency report.
- Added sample ledger rows.

- Kept donations inactive and wallets unpublished.

## v0.4.0 — Read-only Blockchain Explorer Importer Design

Status: implemented as design-only.

- Added read-only public blockchain explorer importer design.
- Added normalized incoming transaction schema.
- Added manual reconciliation workflow.
- Added fictional normalized importer sample rows.
- Added guardrail tests for importer design docs.
- Kept network fetching, private keys, signing, transfers, and exchange APIs out of scope.
