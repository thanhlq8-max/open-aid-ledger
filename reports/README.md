# Reports

Generated and human-reviewed transparency reports should be stored here.

Suggested naming:

```text
YYYY-MM-transparency-report.md
incident-YYYY-MM-DD-short-title.md
wallet-change-YYYY-MM-DD.md
```

Reports must avoid raw private beneficiary data.

## Sample report

`sample-transparency-report.md` is fictional sample data only. It must not be treated as production ledger output or evidence of active donation collection.
