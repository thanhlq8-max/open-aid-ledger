# Campaign: 2026 Open Source Public-Good Support

## Campaign ID

```text
OSS-PUBLIC-GOOD-2026
```

## Purpose

Support maintainers and repositories that provide public utility through open-source software.

## Rationale

Open-source tools can save time, reduce cost, improve research quality, and help communities. It is reasonable for users who benefit from these tools to donate voluntarily.

## Allowed use

- maintainer support;
- hosting costs;
- documentation;
- testing infrastructure;
- security maintenance;
- development tools;
- public demo/report hosting.

## Not allowed

- closed-source private work unless disclosed;
- investment pooling;
- trading signal sale;
- paid manipulation of GitHub stars/forks/issues;
- fake engagement campaigns.

## Reporting

Support to OSS projects should include:

- repository URL;
- maintainer/project name;
- amount;
- tx hash or payment reference;
- purpose note.
