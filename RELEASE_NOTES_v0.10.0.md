# Release Notes — v0.10.0

## Donation-ready candidate review

This release adds the pre-1.0 candidate review layer for Open Aid Ledger.

It is still a dry-run/candidate release only.

```text
DONATIONS_ACTIVE: NO
WALLETS_PUBLISHED: NO
CUSTODY_AUTOMATION: NO
TRADING_USE: FORBIDDEN
RETURN_PROMISE: FORBIDDEN
```

## Added

- Pre-1.0 candidate review document.
- Donation-ready candidate checklist.
- External review request template.
- Candidate decision record template.
- Operational risk acceptance document.
- Candidate review example JSON.
- Candidate validation script.
- Candidate validation tests.

## Not included

- No real wallet address.
- No donation activation.
- No transaction signing.
- No asset transfer automation.
- No exchange account integration.
- No trading or investment feature.
